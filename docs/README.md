# docs/

Design notes, recipes, and media referenced from the top-level `README.md`.

## Files

- **`RECIPES.md`** — end-to-end workflows across BizHawk cores: RAM hunting
  on SNES / NES / N64, frame-precise input, snapshot-experiment-restore,
  cross-system regression testing.
- **`SCOPE.md`** — capability matrix vs. mGBA / PPSSPP / RetroArch / PINE
  bridges. What BizHawk uniquely enables and what's deferred.
- **`SUPER-METROID-ADDRESSES.md`** — curated WRAM addresses for Super Metroid
  (the worked example in the top-level demo). Useful starting point for the
  Samus-driving recipe.
- **`sm-claude-play.gif` / `sm-claude-play.mp4`** — demo media: Claude driving
  Samus through Ceres Station via `bizhawk_play_input_sequence`.
